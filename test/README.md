# Тестовые датасеты

 - *eng* - документы на английском языке
 - *rus* - документы на русском языке
 - *inscription* - короткие фигурные надписи, вывески
 - *image* - картинки, фотографии без текстового содержимого

 Датасеты используются в функциональных тестах в составе проекта, а такде могут быть использованы для проверки сервиса с использованием внешних инструментов (curl, Postman и т.д.)
